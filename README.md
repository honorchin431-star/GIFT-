# 星空下的能量站 - 使用说明

这是一个为你的朋友准备的专属网页礼物。

## 1. 如何预览
双击文件夹中的 `index.html` 文件，使用浏览器（推荐 Chrome, Safari, Edge）打开即可。

## 2. 如何添加背景音乐
1. 准备一首好听的音乐文件（MP3格式）。
2. 将文件重命名为 `bgm.mp3`，并放在与 `index.html` 相同的文件夹中。
3. 或者，用记事本打开 `index.html`，找到第 280 行左右：
   ```html
   <source src="bgm.mp3" type="audio/mpeg">
   ```
   将 `bgm.mp3` 修改为你的音乐文件名。

## 3. 如何修改名字
1. 用记事本或代码编辑器打开 `index.html`。
2. 找到第 373 行左右的变量 `introLetter`：
   ```javascript
   const introLetter = "嗨，[她的名字] ...";
   ```
3. 将 `[她的名字]` 替换为你朋友的真实名字。

## 4. 测试与重置
- **每日一签**：网页会记录日期，每天只能抽一次。如果想测试“新的一天”的效果，可以在浏览器控制台清除 LocalStorage，或者手动修改系统时间。
- **重看开场动画**：如果你想重新体验第一次打开的感觉，请清除浏览器缓存（LocalStorage），或者在浏览器控制台中输入 `localStorage.clear()` 然后刷新。

## 5. 部署
如果你想发给朋友，可以将这个文件夹打包，或者将 `index.html` 和音乐文件上传到任何静态网页托管服务（如 GitHub Pages, Vercel 等）。
